# Contact Manager - Java Swing + MySQL

Application desktop de gestion de contacts développée en Java Swing avec connexion MySQL.

## Fonctionnalités

- Ajouter un contact
- Modifier un contact
- Supprimer un contact
- Afficher les contacts

## Technologies utilisées

- Java
- Swing
- MySQL
- JDBC

## Configuration base de données

Créer la base :

CREATE DATABASE gestion_contacts;

Puis créer la table contacts.
